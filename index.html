The frist try

发表于 2020-05-30

确认过眼神，hexo-admin是我想找的功能，赞呀，在线编辑！

开始使用

npm install --save hexo-admin
hexo server -d
open http://localhost:4000/admin/

首先，安装插件。
再打开服务。
最后打开localhost:4000/admin/即可编辑博文。

编辑之后呢？

保存并上传

admin插件这个网页会自动保存了编辑内容。（直接打开本地的md文档就能看到。）但是并没有直接deploy到博客上，没有上传到GitHub里。
这个时候环顾一下，理所当然点开admin网页导航栏的Deploy。然后蹦出来一个令人迷惑的页面。

然后就Error: Config value “admin.deployCommand” not found了，慌不慌。
不要慌，Deploy之前，还需要编辑配置文件_config.yml。
如果是Windows则在末尾加上

然后在同级目录新建hexo-pubish.bat文件，文件内容如下：

hexo g -d

如果是Linux系统则参考Issues。
编辑完毕后，就可以点击Deploy，直接部署发布Github博客上。
